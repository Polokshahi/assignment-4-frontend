SkillBridge is a simple and modern web app that helps students find and book sessions with expert tutors. This repository contains the frontend built with Next.js and TypeScript.
Project Links

Live Site: https://assignment-4-frontend-six.vercel.app/

Features

Role-based access for Students, Tutors, and Admin Browse tutors by category, price, and rating Real-time booking based on tutor availability Fully responsive UI for mobile, tablet, and desktop

Tech Stack

Next.js (App Router) TypeScript Tailwind CSS & Shadcn UI Lucide Icons Sonner for notifications

Getting Started

Clone the repository git clone Install dependencies npm install

Add environment variables in .env.local

Run the app npm run dev
